    <summary>
        <h2>📁 𝔽ile ℕaming</h2>
    </summary>

>
>Case: snake_case

Examples:
```text
    user_controller.js
    auth_service.ts
    main_layout.html
    global_styles.css
```

<details id="css_convention">
    <summary>
        <h2>🎨 ℂss ℂonvention</h2>
    </summary>

Format: ComponentName-element-subelement  
Case-ComponentName: PascalCase  
Case-element-subelement: kebab-case  

Example:

```scss
    .Card-container {  
        display: flex;  
    }  

    .Card-title {  
        font-size: 1.2rem;  
    }  
```

</details>

<details id="js_convention">
    <summary>
        <h2>⚙️ 𝕁s ℂonvention</h2>
    </summary>

Variables: camelCase  
Functions: camelCase  
Classes: PascalCase  
Constants: UPPER_SNAKE_CASE  

Examples:
```js
let userName;  
const API_URL = "";  

class UserService {}  
```

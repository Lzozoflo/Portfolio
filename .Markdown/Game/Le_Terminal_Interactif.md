# Conception UI/UX pour "Le Terminal Interactif"

Identité Visuelle :

Style : Strictement textuel et monospaced. Classique, évoquant le développement "hardcore".

Palette de Couleurs : Police monospaced de couleur vive (ex: vert matrix, ambre chaleureux) sur un fond noir uni ou gris très foncé. Pas de distraction visuelle.

Structure : Une ligne d'invite de commande (prompt) qui clignote (ex: invite@votreportfolio:~$). Le texte tapé apparaît immédiatement.

Résultats : Les résultats des commandes s'empilent vers le bas.

UX & Interactions :

Feedback de Commande : Les commandes valides s'exécutent avec une vitesse "machine" (instantanément).

Expérience de Découverte : La commande help est essentielle et affiche la liste des mots-clés (ex: bio, contact, projets).

Easter Eggs : Les commandes invalides affichent un message générique poli ("Commande non reconnue"), mais certaines commandes cachées (ex: sudo matrix, idclip) peuvent déclencher une animation visuelle unique (ex: cascade de caractères).

Navigation Facile : Un bouton discret "Fermer le Terminal" ou "Mode Visuel" pour revenir à la navigation normale du portfolio.

Ambiance : Mystérieuse, geek, experte, technique.

# Conception UI/UX pour "Le Snake de Navigation"

Identité Visuelle :

Style : Néo-rétro ou cyberpunk-chic. Pas de "pixel art" grossier, mais des lignes propres et lumineuses.

Le Serpent : Une ligne lumineuse continue (effet néon de votre couleur d'accent) qui s'allonge. La "tête" est une icône simple de curseur.

L'Espace : Un "labyrinthe" virtuel transparent superposé sur votre section "Expérience" ou "Compétences".

La Nourriture (Points à Collecter) : Des icônes lumineuses et floues (effets de "bloom") qui clignotent doucement (ex: une petite pile de livres pour 'formation', une icône de valise pour 'expérience').

UX & Interactions :

Feedback d'Action : Lorsqu'une icône est "mangée", une petite onde de choc visuelle se propage depuis le serpent.

Affichage de l'Info : Juste après avoir collecté une icône, une bulle d'info épurée ou un panneau latéral glisse pour afficher le détail (ex: "3 ANS EXPÉRIENCE DÉVELOPPEUR").

Fin de Jeu : Si le serpent touche un bord ou lui-même, il "s'éteint" doucement. Un bouton "Rejouer" ou "Continuer la Navigation" apparaît.

Ambiance : Dynamique, fluide, technologique.
